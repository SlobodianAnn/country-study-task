Eсть необходимость сделать карту и таблицу с городами и странами. Тебе дали json представления этих стран, теперь тебе нужно сделать из них объекты.

По программе результат такой: я хочу создать объект страны Украины, передав в конструктор его идентификатор "UA"; из этой страны получить столицу как объект страны. Так же, получить "все города" (которые UA) - Оленевку, Урзуф, Лебеин, Старый Добротвир и другие из файла.

Так же, я хочу сделать наоборот - передать массив с именами городов, получить массив городов-объектов, и из них получить страны, а из них - столицы))) Получается карта связей между городами.

1. Создать фабрику объектов. На вход получаем массив из имен городов (как в cities.json). Каждый из созданных объектов должен иметь методы:
- getName (string)
- getGeoPosition (object lat|lng)
- getCountry (object like country  from countries.json)

2. Создать конструктор объектов like country.
- getName (string)
- getNativeName (string)
- getPhone (string)
- getCapital (country like object)
- getCurrency (string)
- getLangs (array)
- getCities (array of city like objects)
- getCityByName (city like object | undefined)
